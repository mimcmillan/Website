Outcomes and events

Precise abstract concepts of outcome, sample space, event, and probability measure help clarify the assumptions in our probability models.

Events: informal

  • An event is a description of something that can take place.
  • An outcome is a complete description of something that can take place.

All outcomes are events. Typically an event is only a partial description. An outcome is an event of the finest grain.

  • After an event takes place, the fact that it took place constitutes information.
  • Information is given and received by communicating the fact that an event took place.

Events: formalized

  • The set of outcomes, also called the sample space, is a set of complete descriptions of all things that can take place.
  • An event is a subset of the set of outcomes.
Infinities

Difficulties with infinity and the continuum mean that sometimes a wild subset is not a valid event.

Notation

  • Write for a typical set of outcomes, for a single outcome.
  • Write or for typical events.
  • Write for the collection of all events.

An outcome is considered an event by considering instead the singleton subset that contains just that one outcome.

In our version of the Monty Hall game, an outcome is a complete description of a run of the experiment. For example: “Martin first chose Card-3 = Goat-2, then the host revealed Card-2 = Goat-1 and offered a second choice, then Martin switched and received Card-1 = Car.” On the other hand, “Martin was offered a second choice” defines a (complex) event. Technically this event is the subset of all complete outcomes in which Martin was offered a second choice.

Probability measure

Probability measure: concept

Each event is associated with a probability of its occurrence, which is a quantity representing its degree of likelihood.

The meaning of this quantity is the subject of controversy.

  • Classical theorists hold that probability derives from the symmetry of a situation (dice, coins, poker)
  • Evidentialists hold that probability relates known evidence to a proposed hypothesis
  • Bayesians hold that probability measures the degrees of belief of persons
  • Frequentists hold that probability derives from the pattern in repeated trials
  • Propensity theorists hold that probability measures a causal power or tendency inhering in objects
